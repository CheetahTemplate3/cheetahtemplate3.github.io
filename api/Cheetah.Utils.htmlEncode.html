Cheetah.Utils.htmlEncode.htmlEncode(s, codes=[['&', '&amp;'], ['<', '&lt;'], ['>', '&gt;'], ['"', '&quot;']])¶

Returns the HTML encoded version of the given string. This is useful to display a plain ASCII text string on a web page.
